Đậu bắp xào tàu hủ ky
Bông cải cắt nhỏ, xào sơ (đừng chín quá)
Xà lách son đất xào
Măng kho cà chua, tàu hủ
# Đậu bắp + Gừng bào sợi+ Ớt bằm/sa tế + sả bằm + hành tím băm + dầu ăn (ít). Xào xong bỏ ngò rí.
# Mướp cắt sợi + Cà rốt cắt sợi + Ớt sừng cắt sợi + cần tây cắt nhuyễn (càng nhỏ càng tốt) + hành tím băm. Xào xong bỏ tiêu, trộn.
# Nấm rơm (ngâm nước xong cắt đôi/bằm) + Đậu hủ trắng cắt nhỏ + 1/4 trái thơm + Hạt nêm + Hành cắt nhỏ + chao đỏ. Kho xong bỏ tiêu.
# Cà rốt cắt sợi xào chín + dầu ăn + hạt nêm + **Bông cải xanh** cắt chừng ngón tay bỏ vào xào sơ, vẫn còn màu xanh đậm, đổ ra dĩa để ko chín quá.
# Cà tím để vỏ cắt lát + mè + nước tương + Trứng gà
# Nấm đùi gà cắt sợi + cà rốt cắt sợi + hành tây + dấm + nước tương. Xào xong còn nóng bỏ hành lá cắt nhỏ + ngò ri.
# Nấm đùi gà cắt hạt lựu + ớt chuông cắt hạt lựu + tàu hủ cắt nhỏ+ dấm + nước tương. Xào xong trộn tiêu.
# Tàu hủ ki xào chín + Bông bí xào vừa chín còn màu xanh đậm. Đổ ra dĩa để ko chín quá
# Ngó sen + cà chua cắt hạt lựu + bắp cải tím cắt sợi + nước mắm.
# Su hào cắt sợi + Nấm tuyết + dấm + bột nêm + nước tương. Xào xong để dĩa để ko chín quá.
# Cà rốt bào sợi xào chín + cà chua cắt hạt lựu + rau mầm. Trộn dấm + đường + hạt nêm.
**Canh**
# Mướp + Nấm hương + rau bồ ngót + tàu hủ trắng cắt nhỏ + hạt nêm + hành cắt nhỏ
# Cà chua cắt nhỏ + nấm bào ngư + giá + thơm (cắt cỡ ngón tay) + thì là bằm + Me (bỏ hạt) + tàu hủ bi + bạc hà + đậu bắp
**Khoai**
# Khoai lang bí cắt sợi + Củ sắn cắt sợi + tàu hủ trắng bóp nhuyễn + hành boa-rô cắt mỏng + dấm + nước tương. Xào xong bỏ rau húng.
# Khoai môn sọ gọt vỏ rồi luộc. Khi luộc chín còn nóng cắt nhỏ + giả nhuyễn + Bột đậu xanh ngâm nước + Nấm mèo cắt nhỏ + củ cải trắng. Xào ròi bỏ tiêu.
